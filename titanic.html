1. Visualisierung: Histogramm

Wann ein Histogramm?

Immer dann, wenn man eine einzelne numerische Variable hat und verstehen will, wie die Werte verteilt sind wo die Masse liegt, ob es Ausreier gibt, ob die Verteilung symmetrisch oder schief ist.

Frage: Wie ist das Alter der Passagiere verteilt?

In[2]:
fig, ax = plt.subplots(figsize=(9, 5))

# Histogramm erstellen:
ax.hist(df['age'], bins=30, color='steelblue', edgecolor='white', linewidth=0.6)

# Median berechnen und visualisieren:
median_age = df['age'].median()
ax.axvline(median_age, color='tomato', linestyle='--', linewidth=2,
           label=f'Median: {median_age:.0f} Jahre')

ax.set_title('Altersverteilung der Titanic-Passagiere', fontsize=15, fontweight='bold', pad=14)
ax.set_xlabel('Alter', fontsize=12)
ax.set_ylabel('Anzahl Passagiere', fontsize=12)
ax.legend(fontsize=11)

sns.despine()
plt.tight_layout()
plt.show()

print(f"Median: {df['age'].median():.0f} Jahre | Mittelwert: {df['age'].mean():.1f} Jahre")
print(f"Jngster: {df['age'].min():.0f} Jahre | ltester: {df['age'].max():.0f} Jahre")

Beobachtung: Die meisten Passagiere waren zwischen 20 und 40 Jahre alt. Es gibt einen kleinen Peak bei Kindern und einige ltere Passagiere als Ausreier.

Wann kein Histogramm? Wenn man Gruppen miteinander vergleichen mchte dafr ist der Boxplot deutlich besser geeignet (s. Visualisierung 3).

2. Visualisierung: Balkendiagramm

Wann ein Balkendiagramm?

Wenn man kategorische Gruppen vergleicht z.B. Klassen, Geschlecht, Herkunft. Balken machen Unterschiede zwischen Kategorien direkt und przise ablesbar.

Frage: Wie unterscheidet sich die berlebensrate nach Reiseklasse und Geschlecht?

In[3]:
survival = df.groupby(['pclass', 'sex'])['survived'].mean().reset_index()
survival['survived_pct'] = survival['survived'] * 100

fig, ax = plt.subplots(figsize=(9, 5))

colors_sex = {'male': 'steelblue', 'female': 'tomato'}
labels_sex = {'male': 'Mnner', 'female': 'Frauen'}
x = [0, 1, 2]
width = 0.35

for i, (sex, grp) in enumerate(survival.groupby('sex')):
    offset = -width / 2 if i == 0 else width / 2
    bars = ax.bar([xi + offset for xi in x], grp['survived_pct'],
                  width=width, label=labels_sex[sex], color=colors_sex[sex], alpha=0.88)
    for bar in bars:
        h = bar.get_height()
        ax.text(bar.get_x() + bar.get_width() / 2, h + 1.2, f'{h:.0f}%',
                ha='center', va='bottom', fontsize=10.5)

ax.set_xticks(x)
ax.set_xticklabels(['1. Klasse', '2. Klasse', '3. Klasse'], fontsize=12)
ax.set_ylabel('berlebensrate (%)', fontsize=12)
ax.set_ylim(0, 115)
ax.set_title('berlebensrate nach Klasse und Geschlecht', fontsize=15, fontweight='bold', pad=14)
ax.legend(fontsize=11)

sns.despine()
plt.tight_layout()
plt.show()

Beobachtung: Frauen berlebten in allen Klassen deutlich hufiger. Klassenunterschiede sind stark erkennbar. Warum kein Tortendiagramm? Tortendiagramme eignen sich nur fr Anteile eines Ganzen mit wenigen Kategorien. Fr Gruppenvergleiche sind Balken fast immer besser Lngen lassen sich viel prziser ablesen als Winkel.

3. Visualisierung: Boxplot

Wann ein Boxplot?

Wenn man eine numerische Variable ber mehrere Gruppen hinweg vergleichen mchte. Der Boxplot zeigt Median, Streuung (IQR) und Ausreier auf einmal ein Histogramm knnte das nur mit mehreren berlagerten Grafiken.

Frage: Unterscheidet sich das Alter zwischen berlebenden und Nicht-berlebenden?

In[4]:
df_box = df.copy()

Beobachtung: Der Median ist bei beiden Gruppen hnlich. Kinder tauchen leicht hufiger unter den berlebenden auf.

Wann kein Boxplot? Bei sehr kleinen Stichproben (< 20 Werte) dann ist ein Strip- oder Swarmplot ehrlicher, weil er die einzelnen Datenpunkte zeigt.

4. Visualisierung: Scatterplot

Wann ein Scatterplot?

Wenn man den Zusammenhang zwischen zwei numerischen Variablen untersucht. Durch Farbe kann man gleichzeitig eine dritte (kategorische) Variable einbinden so werden drei Dimensionen in einer Grafik sichtbar.

Frage: Gibt es einen Zusammenhang zwischen Alter, Fahrpreis und berleben?

In[5]:
df_scatter = df.copy()
df_scatter['berleben'] = df_scatter['survived'].map({0: 'Nicht berlebt', 1: 'berlebt'})

Wann kein Scatterplot? Bei kategorischen Variablen auf beiden Achsen dann lieber eine Heatmap oder ein Balkendiagramm.

Anhang Visualisierung 5: Heatmap

Korrelationsmatrix der numerischen Variablen

Wann eine Heatmap?
Wenn man schnell einen berblick ber Zusammenhnge zwischen vielen Variablen bekommen mchte. Besonders ntzlich am Anfang einer explorativen Analyse als Orientierungshilfe, bevor man sich auf einzelne Variablen fokussiert.

In[6]:
cols = ['survived', 'pclass', 'age', 'sibsp', 'parch', 'fare']
corr = df[cols].corr()

fig, ax = plt.subplots(figsize=(7, 5))

sns.heatmap(corr, annot=True, fmt='.2f', cmap='coolwarm', center=0,
            linewidths=0.5, square=True, ax=ax, annot_kws={'size': 10})

ax.set_title('Korrelationsmatrix  Titanic-Merkmale', fontsize=14, fontweight='bold', pad=14)
plt.tight_layout()
plt.show()

print("Strkste Korrelationen mit 'survived':")
print(corr['survived'].drop('survived').sort_values(key=abs, ascending=False).to_string())

Anhang Visualisierung 6: Pairplot

Alle numerischen Variablen auf einen Blick

Wann ein Pairplot?
Nur in der frhen Explorationsphase, wenn man noch gar nicht wei, welche Variablen interessant sind. Fr Berichte oder Prsentationen ist er meist zu unbersichtlich er ist ein Werkzeug fr den Analysten, nicht fr das Publikum.

In[7]:
df_pair = df[['survived', 'age', 'fare', 'pclass']].copy()
df_pair['berleben'] = df_pair['survived'].map({0: 'Nicht berlebt', 1: 'berlebt'})

g = sns.pairplot(
    df_pair[['age', 'fare', 'pclass', 'berleben']],
    hue='berleben',
    palette={'Nicht berlebt': 'steelblue', 'berlebt': 'tomato'},
    diag_kind='hist',
    plot_kws={'alpha': 0.4, 's': 18}
)

g.figure.suptitle('Pairplot  berblick aller numerischen Variablen',
                   y=1.02, fontsize=13, fontweight='bold')
plt.tight_layout()
plt.show()

Fazit: Der Pairplot ist ein Tool fr die eigene Analyse nicht fr die Kommunikation von Ergebnissen. Sobald man wei, was interessant ist, greift man auf gezielte Einzelplots zurck.

Zusammenfassung:

Ziel Visualisierung
Verteilung einer numerischen Variable Histogramm
Unterschiede nach Kategorien Balkendiagramm
Verteilung zwischen Gruppen vergleichen Boxplot
Zusammenhnge zwischen zwei Variablen erkennen Scatterplot
Korrelation zweier Variablen Heatmap
Interessante Variablen entdecken Pairplot (nur Exploration)
