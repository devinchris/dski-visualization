# Visualisierung in der EDA – Anhang zur Präsentation

**Modul:** Data Science & KI

Das Repo enthält den interaktiven Anhang zur Präsentation über Datenvisualisierung.  
Als Datensatz wird der *Titanic-Datensatz* verwendet.

---

## Dateien

| Datei | Beschreibung | Öffnen mit |
|---|---|---|
| `titanic.ipynb` | Jupyter Notebook mit vollständigem Code und Visualisierungen | `jupyter notebook titanic.ipynb` |
| `titanic.html` | Statische HTML-Version des Notebooks | Einfach im Browser öffnen (Doppelklick) |
| `titanic.pdf` | PDF-Version zum schnellen Lesen | PDF-Viewer / Browser |

---

## Schnellstart (Notebook)

```bash
# Abhängigkeiten installieren (einmalig)
pip install jupyter pandas matplotlib seaborn

# Notebook starten
jupyter notebook titanic.ipynb
```

> **Kein Python installiert?** Einfach `titanic.html` im Browser öffnen – kein Setup nötig.

### Dependencies
- jupyter
- pandas
- matplotlib
- seaborn
